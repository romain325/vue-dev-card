# vue-dev-card

## Project setup
```
yarn add vue-dev-card
```
## How to use

```html
<template>
  <vue-dev-card>
    <git-hub username="romain325"/>
    <readme username="romain325"/>
  </vue-dev-card>
</template>
```

You can implement your own card and use it on your personal project or create a Merge Request, I'll be happy to review it
